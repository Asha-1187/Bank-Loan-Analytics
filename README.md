# Bank-Loan-Analytics
Project Overview  This project focuses on analyzing bank loan data to understand customer behavior, loan performance, and risk factors. The goal is to generate actionable insights that help financial institutions make data-driven lending decisions.

Objectives:
Analyze loan approval and rejection trends
Evaluate default risk and repayment behavior
Identify patterns in customer demographics and loan grades
Monitor regional and time-based loan performance

🛠️ Tools & Technologies
SQL
Excel
Power BI
Tableau

📈 Key KPIs & Analysis
Year-wise Loan Amount Trends
Grade & Sub-grade wise Revolving Balance (revol_bal)
Total Payment Analysis (Verified vs Non-Verified Customers)
State-wise and Month-wise Loan Status
Home Ownership vs Last Payment Date analysis

📊 Dashboard Features
Interactive filters for state, grade, and time period
Dynamic KPIs for loan performance and risk analysis
Visualizations including bar charts, line charts, and pie charts
Drill-down capabilities for detailed insights
🔍 Key Insights

Verified customers show better repayment behavior compared to non-verified customers
Certain loan grades have higher default risk
Loan performance varies significantly across states and months
Home ownership has a noticeable impact on repayment patterns
